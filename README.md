# Smart Study Planner & Productivity Dashboard

## Overview
The Smart Study Planner and Productivity Dashboard is an easy-to-use tool that helps students organize and plan their academic work. It also tracks progress through visual analytics. This planner improves consistency, time management, and discipline using an effective productivity tracker.

---

## Features
- Add, edit, and delete study tasks  
- Categorize tasks into four groups: Assignment, Revision, Reading, and Exam Preparation  
- Set deadlines, priority levels, and task duration  
- Mark tasks as completed  
- Productivity Dashboard includes:
  - Daily completion rate
  - Weekly analytics
  - Streak tracking
  - Visual charts  
- User-friendly interface  

---

## Technologies Used
- Python
- Matplotlib
- Datetime library for task scheduling

---

## Installation & Running Instructions

### 1. Download the project
- Open this GitHub repository.
- Click the green Code button.
- Select Download ZIP and extract it on your computer.

### 2. Run the application
- Open the extracted folder.
- Go to the src folder.
- Double-click main.py to run the application.
  (or open Command Prompt in the folder and type: python main.py)

### 3. Requirements
- Ensure Python is installed on your system.
- If any library is missing, install it using:
  pip install -r requirements.txt

---

## Testing Instructions
- Add sample tasks and check if they appear in the planner  
- Edit and delete tasks to confirm updates  
- Mark tasks as completed and check the analytics dashboard  
- Test with incorrect inputs (empty title, wrong date format)  
- Verify that the streak count updates correctly
